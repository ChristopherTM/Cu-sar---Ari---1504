# 🌌 Proyecto: Cuásar Ari - 1504

### Registro de Anomalía de Alto Redshift
**Investigador:** Christopher Téllez Monroy

## 🔭 Resumen Técnico
Este repositorio contiene el análisis espectroscópico y cosmológico del cuásar **Ari - 1504**, localizado mediante el Sloan Digital Sky Survey (SDSS). El objeto representa una fuente de energía masiva del universo temprano.

| Parámetro | Valor |
| :--- | :--- |
| **Designación** | Cuásar Ari - 1504 |
| **ObjID** | 1237648721215816394 |
| **Redshift (Z)** | 3.589 |
| **Coordenadas (J2000)** | RA 138.07168, Dec +02.02231 |
| **Distancia (DL)** | 31,513.17 Mpc |
| **Luminosidad Ly-α** | ~1.04e8 L⊙ |

## 🛠️ Metodología
El análisis se realizó utilizando `Astroquery` y `Astropy` sobre el servidor de datos del SDSS. Se integró el flujo de la línea de emisión Lyman-Alpha para determinar la potencia energética bolométrica parcial.
